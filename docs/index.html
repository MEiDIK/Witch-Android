Witch is a flexible view-data binding library for Android.

  • - Bind data to views with @annotations.

  • - Compose functional bind flows.

  • - View holders built in.

Simple

Bind values directly to views without any lookup. 

            
@BindToTextView(id= R.id.title)
String title = “Hello Witch!”;
...

Witch.bind(model, activity);

Flexible

Views not backed up by an annotation, can be bound with custom view class. 

            
@BindToView(id = R.id.address, class = AddressView.class, set= "address")
String address = "Misty mountains";

Annotate methods for dynamic values 

            
@BindToTextView(R.id.name)
void name() {
  if(user == null) {
    return "Magica de Hex";
  }
  return user.givenName + " " + user.familyName;
}

More advanced bindings can be create with Binder. 

            
Binder binder = Binder.create(new SyncOnBind<ImageView, String>() {
  @Override 
  void onBind(ImageView view, String url) {
    Picasso
    .with(view.getContext())
    .load(url)
    .into(imageView);
  }
});

// Use ValueBinder to use binder with annotation
@BindTo(R.id.image)
ValueBinder kitten = ValueBinder.create(“www.snylt.se/kitten.png”, binder);

Dynamic

Bind functions can be re-used and composed in chains. 

            
Binder
.create(new PicassoLoadOnBind<ImageView, String>()) 
.next(new InvisibleIfNull<ImageView, String>());
